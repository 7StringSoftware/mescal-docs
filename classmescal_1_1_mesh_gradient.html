Detailed Description

A mesh gradient is a gradient with colors that transition smoothly between a set of points. Mesh gradient colors can blend and flow in multiple directions simultaneously.

A MeshGradient creates and stores a set of points called vertices. Each vertex has a designated color. A group of four vertices defines a patch. Each edge of the patch is a cubic Bezier spline.
